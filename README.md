# 🧊 Balance on Ice

## About
A small Unity prototype made to understand how visual novels work.

This is a **quick practice project**, not a full game.

## Purpose
- Learn basics of visual novel structure  
- Practice UI and dialogue systems  
- Experiment with simple mechanics
